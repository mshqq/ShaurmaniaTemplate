## Структура блоков

| #   | Блок                | Описание                                                      |
| --- | ------------------- | ------------------------------------------------------------- |
| 1   | **Шапка (nav)**     | Логотип, телефон, кнопка «Заказать звонок»                    |
| 2   | **Слайдер**         | Карусель с преимуществами, кнопки prev/next, точки-индикаторы |
| 3   | **Особенности**     | Три карточки с иконками — свежесть, соусы, открытая кухня     |
| 4   | **Меню + фильтр**   | Сетка карточек товаров с фильтрацией по категориям            |
| 5   | **О нас**           | Фото + текст о компании                                       |
| 6   | **Форма**           | Сведения о компании + подписка (имя, e-mail, кнопка)          |
| 7   | **Футер**           | Логотип, копирайт, список из 13 адресов точек                 |
| 8   | **Кнопка «Наверх»** | Фиксированная кнопка возврата в начало страницы               |

## Запуск

Статический сайт, серверная часть не нужна. Достаточно открыть `index.html` в браузере.

Деплой на GitHub Pages настроен через `.github/workflows/static.yml`.

---

## Учебный контекст

Проект выполнен по заданию учебной практики. Цель — создать сайт-визитку предприятия в формате Landing Page со следующими обязательными блоками: шапка, слайдер, особенности услуг, каталог с фильтром, информация о компании, форма подписки, кнопка «Наверх».

Оценка выставляется по методике B = B1 + B2 + B3 + B4 + B5 (дизайн, кодирование, содержание, грамматика, привлекательность; максимум 25 баллов).
